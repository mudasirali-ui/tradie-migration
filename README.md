## Mazdoor Migration – React (Vite) app

This project is a single-page React application for **Mazdoor Migration**, a migration and licensing pathway platform connecting overseas electricians and tradespeople with Australian employers and training providers.

### Scripts

- `npm install` – install dependencies
- `npm run dev` – start the development server
- `npm run build` – create a production build
- `npm run preview` – preview the production build locally

### Tech stack

- React (Vite)
- React Router (client-side routing)
- Hand-crafted responsive CSS with gradients, cards, and animations

### Key routes

- `/` – Marketing homepage with hero, CTAs for tradies and employers
- `/otsr` – Detailed Offshore Technical Skills Record (OTSR) information page
- `/tradies` – Overview of the pathway for overseas electricians/tradespeople
- `/employers` – Value proposition for Australian employers
- `/about` – High-level about page
- `/contact` – Static contact form (demo; not wired to any backend)
